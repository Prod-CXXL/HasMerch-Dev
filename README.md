HasMerch

HasMerch is a creator-focused merchandise platform that enables creators, organizations, recruiters, businesses, and communities to create and sell custom merchandise.

Current Features

* Product catalog
* Product detail pages
* Shopping cart integration
* Cloudflare Pages hosting
* GitHub version control

Planned Features

* Creator storefronts
* Marketplace functionality
* Stripe Checkout integration
* Stripe Connect payouts
* Affiliate system
* Analytics dashboard
* Digital products
* Mobile responsive design

Technology Stack

Frontend

* HTML5
* CSS3
* JavaScript

Hosting

* Cloudflare Pages
* GitHub

Payments

* Stripe Checkout (planned)
* Stripe Connect (planned)

Development Workflow

1. Make changes locally
2. Test changes
3. Commit to Git
4. Push to GitHub
5. Cloudflare automatically deploys

Repository Structure

* /website - Website files
* /assets - Images and media
* /docs - Documentation
* /data - Product and configuration data
